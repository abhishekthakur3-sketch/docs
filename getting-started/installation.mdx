## Step 1: Install the Package

<CodeGroup>
```bash npm
npm install @tarmac/design-system
```

```bash yarn
yarn add @tarmac/design-system
```

```bash pnpm
pnpm add @tarmac/design-system
```
</CodeGroup>

## Step 2: Install Peer Dependencies

TARMAC requires React 18+:

```bash
npm install react@^18.0.0 react-dom@^18.0.0
```

## Step 3: Import Styles

Add the TARMAC stylesheet to your app entry point:

```tsx
// App.tsx or index.tsx
import '@tarmac/design-system/styles.css';
```

## Step 4: Wrap with Provider

Wrap your app with the TARMAC provider for theming and context:

```tsx
import { TarmacProvider } from '@tarmac/design-system';

function App() {
  return (
    <TarmacProvider>
      {/* Your app content */}
    </TarmacProvider>
  );
}
```

## Requirements

| Requirement | Version |
|-------------|---------|
| React | 18.0+ |
| React DOM | 18.0+ |
| Node.js | 16.0+ |
| TypeScript | 4.7+ (optional) |

<Info>
  TARMAC is built with TypeScript and includes type definitions out of the box. No additional `@types` packages needed.
</Info>
