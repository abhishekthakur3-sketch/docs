## Overview

Carousel allows users to navigate through a sequence of items using swipe, scroll, or arrow controls. It adapts its layout across screen sizes and displays position via dot indicators.

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Keyboard | Arrow keys navigate between slides |
| Focus | Each slide and control must be focusable |
| ARIA | Use `role="region"` with `aria-label="Carousel"` |
| Indicators | Announce current position (e.g. "Slide 1 of 5") |
| Controls | Arrow buttons must have `aria-label="Previous"` / `"Next"` |
| Auto-play | Must have a pause control if auto-play is enabled |

## Developer Handoff

<CodeGroup>
```tsx Component API
interface CarouselProps {
  breakpoint?: 'mobile' | 'desktop-small' | 'desktop-large';
  header?: string;
  subtext?: string;
  addonSlot?: React.ReactNode;
  indicators?: boolean;
  items: React.ReactNode[];
  defaultIndex?: number;
  onChange?: (index: number) => void;
}
```

```tsx Usage
import { Carousel } from '@tarmac/design-system';

// Mobile carousel
<Carousel
  breakpoint="mobile"
  header="Featured Items"
  subtext="Browse our latest collection"
  indicators={true}
  items={[
    <ProductCard key="1" />,
    <ProductCard key="2" />,
    <ProductCard key="3" />,
  ]}
/>

// Desktop with arrow navigation
<Carousel
  breakpoint="desktop-large"
  header="Recommended"
  indicators={true}
  items={[
    <BannerCard key="1" />,
    <BannerCard key="2" />,
  ]}
  onChange={(index) => trackSlideView(index)}
/>
```
