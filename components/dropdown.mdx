## Overview

Dropdowns combine an input field and an options list to create a complete selection interface. Users interact by expanding the list and selecting an option — the dropdown then collapses and displays the chosen value. The system is built from three foundational elements: Dropdown Input Field, Dropdown List Cells, and Dropdown Lists.

## Sub-Component: Dropdown Input Field

The visible control that triggers the dropdown list. Displays the currently selected option and allows users to open the list.

| Property | Options | Description |
|----------|---------|-------------|
| Style | Regular, Blue | Visual theme alignment |
| Type | Default, Add-on Left, Add-on Right | Layout configuration |
| State | Default, Hover, Active, Focused, Disabled, Ghost | Interaction feedback |
| Size | Large, Default, Small | Dimensional scale |

---

## Sub-Component: Dropdown List Cells

Individual selectable options within the dropdown list. Each cell acts as a selectable item with optional visual elements.

| Property | Options | Description |
|----------|---------|-------------|
| Style | Regular, Info Blue | Visual differentiation |
| State | Default, Hover, Pressed, Disabled | Selection feedback |
| Size | Large, Default, Small | Matches input field size |

---

## Sub-Component: Dropdown Lists

Container holding a group of dropdown list cells. Appears when the input field is expanded.

| Property | Options | Description |
|----------|---------|-------------|
| List Count | 2–8 Cells | Number of visible options |
| Variant | List Set 1, List Set 2 | Cell composition pattern |
| Size | Large, Default, Small | Matches input field size |

---

## Input Field Types

| Type | Description | Use When |
|------|-------------|----------|
| Default | Standard dropdown with trailing indicator | General selection fields |
| Add-on Left | Container on left for icons or context | Category-prefixed selections |
| Add-on Right | Container on right for controls or actions | Selections with secondary actions |

## States

| State | Visual Treatment | Behavior |
|-------|-----------------|----------|
| Default | Neutral border, placeholder text | Ready for interaction |
| Hover | Subtle border/background change | Cursor over the field |
| Active | Expanded list visible | User clicked to open |
| Focused | Focus ring visible | Keyboard navigation active |
| Disabled | 40% opacity, non-interactive | Selection unavailable |
| Ghost | Skeleton/loading appearance | Data is loading |

---

## Behavior

| Interaction | Result |
|-------------|--------|
| Click input field | Expands dropdown list |
| Select an option | Updates field value, collapses list |
| Click outside | Collapses list without selection |
| Arrow keys (when focused) | Navigate between options |
| Enter / Space | Select highlighted option |
| Escape | Close dropdown list |
| Disabled state | Cannot be opened |

---

## List Count

Dropdown lists support 2–8 visible cells. Keep lists concise for readability.

| Count | Cells | Recommended Use |
|-------|-------|-----------------|
| 2 | 2 options | Binary-plus selections |
| 3 | 3 options | Small option sets |
| 4 | 4 options | Medium option sets |
| 5 | 5 options | Default recommendation |
| 6 | 6 options | Extended option sets |
| 7 | 7 options | Large option sets |
| 8 | 8 options | Maximum before scrolling |

## Usage Guidelines

<CardGroup cols={2}>
  <Card title="✓ Do" icon="check" color="#16A34A">
    - Use for selecting from predefined option sets
    - Keep option labels short and scannable
    - Match dropdown size to surrounding form elements
    - Use Info Blue cells to highlight recommended options
    - Provide a clear placeholder or default selection
  </Card>
  <Card title="✗ Don't" icon="xmark" color="#DC2626">
    - Don't use for binary choices (use Toggle or Checkbox)
    - Don't use for large datasets (use searchable autocomplete)
    - Don't exceed 8 visible options without scrolling
    - Don't nest dropdowns inside other dropdowns
    - Don't use for frequently used quick actions
  </Card>
</CardGroup>

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Role | Use `role="listbox"` on the options container |
| Keyboard | Arrow keys navigate options, Enter/Space selects, Escape closes |
| Focus | Visible focus ring on input field and active option |
| Screen reader | Announce selected value, expanded/collapsed state, option count |
| Disabled | Communicate disabled state via `aria-disabled="true"` |
| Label | Associate with `<label>` or `aria-label` for context |

## Developer Handoff

<CodeGroup>
```tsx API
interface DropdownProps {
  style?: 'regular' | 'blue';
  type?: 'default' | 'addon_left' | 'addon_right';
  size?: 'lg' | 'md' | 'sm';
  state?: 'default' | 'hover' | 'active' | 'focused' | 'disabled' | 'ghost';
  placeholder?: string;
  value?: string;
  options: DropdownOption[];
  onChange?: (value: string) => void;
  disabled?: boolean;
  label?: string;
  helperText?: string;
  leadingIcon?: ReactNode;
  trailingIcon?: ReactNode;
}

interface DropdownOption {
  value: string;
  label: string;
  subtext?: string;
  avatar?: string;
  leadingIcon?: ReactNode;
  trailingIcon?: ReactNode;
  disabled?: boolean;
  cellStyle?: 'regular' | 'info_blue';
}
```

```tsx Usage
// Basic dropdown
<Dropdown
  label="Select country"
  placeholder="Choose an option"
  options={[
    { value: 'in', label: 'India' },
    { value: 'us', label: 'United States' },
    { value: 'uk', label: 'United Kingdom' },
  ]}
  onChange={(val) => console.log(val)}
/>

// With add-on left
<Dropdown
  type="addon_left"
  size="lg"
  label="Category"
  options={categoryOptions}
/>

// With avatars and subtext
<Dropdown
  placeholder="Assign to"
  options={[
    { value: '1', label: 'John Doe', subtext: 'Engineering', avatar: '/avatars/john.png' },
    { value: '2', label: 'Jane Smith', subtext: 'Design', avatar: '/avatars/jane.png' },
  ]}
/>

// Info Blue highlighted option
<Dropdown
  options={[
    { value: 'rec', label: 'Recommended', cellStyle: 'info_blue' },
    { value: 'std', label: 'Standard' },
    { value: 'eco', label: 'Economy' },
  ]}
/>

// Disabled
<Dropdown disabled placeholder="Not available" options={[]} />
```
