## Overview

Input Field Stepper combines multiple input fields into a visually connected sequential flow. Common use cases include pickup/delivery address entry and multi-stop route planning.

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Labels | Each input field must have a visible label (e.g. Pickup from, Delivery to) |
| Keyboard | Tab between fields, Enter to confirm, Escape to clear |
| Drag reorder | Provide keyboard alternative for reordering (e.g. move up/down buttons) |
| Add/Remove | Buttons must have `aria-label` (e.g. "Add stop", "Remove stop 2") |
| Focus | Maintain logical focus order through the step sequence |
| Screen readers | Announce step position (e.g. "Stop 1 of 3") |

## Developer Handoff

<CodeGroup>
```tsx Component API
interface InputFieldStepperProps {
  variant?: 'default' | 'entered' | 'multi-stop' | 'multi-stop-2' | 'multi-stop-3';
  showAddButton?: boolean;
  onAddStop?: () => void;
  onRemoveStop?: (index: number) => void;
  onReorder?: (fromIndex: number, toIndex: number) => void;
}

interface StepperInputFieldProps {
  state?: 'default' | 'hover' | 'active' | 'entered' | 'disabled';
  label?: string;
  value?: string;
  subtext?: string;
  showDragHandle?: boolean;
  showCloseIcon?: boolean;
  onClear?: () => void;
  onChange?: (value: string) => void;
}
```

```tsx Usage
import { InputFieldStepper, StepperInputField } from '@tarmac/design-system';

// Default 2-stop stepper
<InputFieldStepper variant="default">
  <StepperInputField label="Pickup from" state="default" />
  <StepperInputField label="Delivery to" state="default" />
</InputFieldStepper>

// Multi-stop with entered values
<InputFieldStepper
  variant="multi-stop"
  onAddStop={() => addNewStop()}
  onRemoveStop={(i) => removeStop(i)}
>
  <StepperInputField label="Pickup from" value="Warehouse A" state="entered" />
  <StepperInputField label="Stop 1" value="Customer B" state="entered" />
  <StepperInputField label="Delivery to" value="Customer C" state="entered" />
</InputFieldStepper>
```
