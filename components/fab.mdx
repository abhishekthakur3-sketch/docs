## Overview

FAB (Floating Action Button) serves as the main entry point for primary actions. It floats above the UI in a fixed position and can expand to reveal a set of related actions.

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Labels | Icon-only FAB must have `aria-label` describing the action |
| Keyboard | Tab to focus, Enter/Space to activate |
| Focus ring | Visible 2px focus outline |
| Touch target | Minimum 44×44px (Large size recommended) |
| Menu | FAB menu must be keyboard navigable with Escape to close |
| Screen readers | Announce expanded/collapsed state via `aria-expanded` |

## Developer Handoff

<CodeGroup>
```tsx Component API
interface FABProps {
  type?: 'default' | 'extended';
  theme?: 'light' | 'dark' | 'info-blue';
  size?: 'large' | 'medium' | 'small';
  state?: 'default' | 'hover' | 'focus' | 'pressed' | 'disabled' | 'ghost';
  icon: React.ReactNode;
  label?: string;           // Required for extended type
  onClick?: () => void;
  'aria-label'?: string;    // Required for default type
}

interface FABMenuProps {
  theme?: 'light' | 'dark' | 'info-blue';
  items: { icon: React.ReactNode; label: string; onClick: () => void }[];
  defaultExpanded?: boolean;
}
```

```tsx Usage
import { FAB, FABMenu } from '@tarmac/design-system';

// Default icon-only FAB
<FAB
  type="default"
  theme="light"
  size="large"
  icon={<AddIcon />}
  aria-label="Add new item"
  onClick={() => handleAdd()}
/>

// Extended FAB with label
<FAB
  type="extended"
  theme="dark"
  size="large"
  icon={<AddIcon />}
  label="Add Item"
  onClick={() => handleAdd()}
/>

// FAB with expandable menu
<FABMenu
  theme="light"
  items={[
    { icon: <ShareIcon />, label: 'Share', onClick: handleShare },
    { icon: <EditIcon />, label: 'Edit', onClick: handleEdit },
    { icon: <DeleteIcon />, label: 'Delete', onClick: handleDelete },
  ]}
/>
```
