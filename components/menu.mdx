## Overview

Menu provides a contextual list of actions or navigation options. It supports single and dual column layouts, nested sublists, and multiple sizes for flexible use across the interface.

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Role | Use `role="menu"` on the container and `role="menuitem"` on each item |
| Keyboard | Arrow keys to navigate, Enter to select, Escape to close |
| Focus | First item receives focus on open; focus trap within menu |
| Submenu | Arrow Right opens submenu, Arrow Left closes it |
| Disabled | Disabled items must have `aria-disabled="true"` and skip focus |
| Screen readers | Announce menu label, item count, and current position |

## Developer Handoff

<CodeGroup>
```tsx Component API
interface MenuProps {
  style?: 'single' | 'dual';
  size?: 'large' | 'default' | 'small';
  title?: string;
  items: MenuItem[];
  onSelect?: (item: MenuItem) => void;
  onClose?: () => void;
}

interface MenuItem {
  label: string;
  icon?: React.ReactNode;
  addon?: 'icon' | 'dot' | 'avatar' | 'checkbox';
  arrow?: 'down' | 'up' | 'right';
  disabled?: boolean;
  checked?: boolean;
  subItems?: MenuItem[];
  onClick?: () => void;
}
```

```tsx Usage
import { Menu } from '@tarmac/design-system';

// Single menu
<Menu
  style="single"
  size="large"
  title="Actions"
  items={[
    { label: 'Edit', icon: <EditIcon /> },
    { label: 'Duplicate', icon: <CopyIcon /> },
    { label: 'Delete', icon: <TrashIcon />, disabled: true },
  ]}
  onSelect={(item) => handleAction(item)}
/>

// Dual menu with sublists
<Menu
  style="dual"
  size="default"
  title="Categories"
  items={[
    {
      label: 'Shipping',
      arrow: 'right',
      subItems: [
        { label: 'Express' },
        { label: 'Standard' },
      ],
    },
    {
      label: 'Returns',
      arrow: 'right',
      subItems: [
        { label: 'Refund' },
        { label: 'Exchange' },
      ],
    },
  ]}
/>
```
