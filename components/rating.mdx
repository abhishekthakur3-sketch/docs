## Overview

The Rating component communicates qualitative feedback through a familiar star-based pattern. It enables users to quickly understand ratings or provide input in an intuitive way. The system is composed of Rating Star (the individual unit) and Rating Set (a group of stars representing a value). It supports fractional ratings and works across both display and input use cases.

## Sub-Components

| Component | Description |
|-----------|-------------|
| Rating Star | Individual star unit with Full, Half, or Empty fill state |
| Rating Set | Collection of 5 stars representing a complete rating value |

---

## Star Variants

| Variant | Visual | Description |
|---------|--------|-------------|
| Full | Filled star | Fully selected value |
| Half | Half-filled star | Partial/fractional value |
| Empty | Outlined star | Unselected value |

## Behavior

| Mode | Description |
|------|-------------|
| Display | Shows a static rating value, supports fractional values via half stars, non-interactive |
| Input | Users select a rating via click or tap, hover previews rating on web, supports half-step interactions if enabled |

---

## Usage Guidelines

<CardGroup cols={2}>
  <Card title="Do" icon="check" color="#16A34A">
    - Use a consistent number of stars (5)
    - Provide clear visual distinction between fill states
    - Include text labels for accessibility
    - Use half stars only when meaningful
    - Pair with numeric value if precision is important
  </Card>
  <Card title="Don't" icon="xmark" color="#DC2626">
    - Don't mix different rating scales within the same product
    - Don't rely only on star visuals without context
    - Don't make interactive areas too small
    - Don't overcomplicate with too many states
    - Don't use ratings for non-feedback purposes
  </Card>
</CardGroup>

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Role | Use `role="img"` for display, `role="radiogroup"` for input |
| Label | Provide `aria-label` with numeric value (e.g., "4.5 out of 5 stars") |
| Keyboard | Arrow keys to change rating in input mode |
| Focus | Visible focus ring on interactive stars |
| Screen reader | Announce current rating value and total scale |

## Developer Handoff

<CodeGroup>
```tsx API
interface RatingProps {
  value?: number;
  max?: number;
  size?: 'lg' | 'md' | 'sm';
  readOnly?: boolean;
  onChange?: (value: number) => void;
  allowHalf?: boolean;
  className?: string;
}
```

```tsx Usage
// Display rating (read-only)
<Rating value={4.5} size="lg" readOnly />

// Input rating
<Rating value={3} size="md" onChange={(val) => setRating(val)} />

// Half-star support
<Rating value={3.5} allowHalf readOnly />

// Small compact rating
<Rating value={4} size="sm" readOnly />

// Empty state
<Rating value={0} size="lg" readOnly />
```
</CodeGroup>
