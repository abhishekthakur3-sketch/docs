## Overview

OTP Fields provide a focused, accessible input experience for one-time password verification. Each digit is entered in a separate box, guiding users through the input process with clear visual feedback.

<CardGroup cols={2}>
  <Card title="✓ Do" icon="check" color="#16A34A">
    - Auto-advance focus to the next field after each digit
    - Auto-submit when all digits are filled
    - Show a resend option with a countdown timer
    - Use Error style immediately on incorrect OTP
    - Support paste to fill all fields at once
  </Card>
  <Card title="✗ Don't" icon="xmark" color="#DC2626">
    - Don't use for non-numeric inputs
    - Don't require manual submission after all digits are entered
    - Don't hide the error message — always show helper text
    - Don't disable the resend option without a visible timer
    - Don't use more than 6 digits
  </Card>
</CardGroup>

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Labels | Group must have a visible label or `aria-label` |
| Keyboard | Tab moves between fields; Backspace clears and moves back |
| Auto-advance | Focus moves automatically after each digit entry |
| Paste support | Pasting a full OTP should populate all fields |
| Screen readers | Announce field position (e.g. "Digit 1 of 4") |
| Error state | Error message must be announced via `aria-live` |
| Contrast | Digit text and border must meet 4.5:1 contrast ratio |

## Developer Handoff

<CodeGroup>
```tsx Component API
interface OtpFieldsProps {
  type?: '4-digit' | '6-digit';
  size?: 'large' | 'medium' | 'small';
  style?: 'default' | 'success' | 'error' | 'info';
  state?: 'default' | 'hover' | 'active' | 'focused' | 'disabled' | 'ghost';
  title?: string;
  titleIcon?: boolean;
  helperTextTop?: string;
  subtext?: string;
  helperTextBottom?: string;
  value?: string;
  onChange?: (value: string) => void;
  onComplete?: (value: string) => void;
}
```

```tsx Usage
import { OtpFields } from '@tarmac/design-system';

// 4-digit default OTP
<OtpFields
  type="4-digit"
  size="large"
  style="default"
  title="Enter OTP"
  helperTextTop="Sent to +91 98765 43210"
  subtext="Resend in 00:30"
  onComplete={(otp) => verifyOtp(otp)}
/>

// 6-digit with error
<OtpFields
  type="6-digit"
  size="large"
  style="error"
  title="Enter OTP"
  helperTextBottom="Incorrect OTP. Please try again."
  onComplete={(otp) => verifyOtp(otp)}
/>

// 4-digit success state
<OtpFields
  type="4-digit"
  size="medium"
  style="success"
  title="OTP Verified"
  helperTextBottom="Your OTP has been verified successfully."
/>

// Disabled
<OtpFields
  type="6-digit"
  size="large"
  state="disabled"
/>
```
