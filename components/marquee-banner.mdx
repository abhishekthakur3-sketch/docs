## Overview

Marquee Banner shows continuously moving text to highlight alerts or announcements without interrupting user flow. It supports light and dark themes across web and mobile devices.

## Accessibility

| Requirement | Guideline |
|-------------|-----------|
| Pause control | Must pause on hover (web) or provide a pause button |
| Speed | Scrolling speed must allow comfortable reading |
| Screen readers | Provide static `aria-label` with the full message text |
| Contrast | Text must meet 4.5:1 contrast ratio against background |
| Reduced motion | Respect `prefers-reduced-motion` — show static text instead |
| Focus | Banner must not steal focus from interactive elements |

## Developer Handoff

<CodeGroup>
```tsx Component API
interface MarqueeBannerProps {
  theme?: 'light' | 'dark';
  type?: 'start' | 'end';
  device?: 'web' | 'mobile';
  messages: string[];
  speed?: number;          // pixels per second
  pauseOnHover?: boolean;  // default: true
  'aria-label'?: string;   // static text for screen readers
}
```

```tsx Usage
import { MarqueeBanner } from '@tarmac/design-system';

// Light theme alert banner
<MarqueeBanner
  theme="light"
  type="start"
  messages={[
    'Be alert: A targeted phishing scam is circulating via SMS',
    'Please do not transfer money to unverified payment links',
  ]}
  pauseOnHover={true}
  aria-label="Security alert: Beware of phishing scams via SMS"
/>

// Dark theme for urgent alerts
<MarqueeBanner
  theme="dark"
  type="start"
  messages={[
    'Service disruption in North India due to weather conditions',
    'Deliveries may be delayed by 24-48 hours',
  ]}
/>
```
</CodeGroup>
