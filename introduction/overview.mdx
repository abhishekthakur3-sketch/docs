## Why TARMAC?

<AccordionGroup>
  <Accordion title="Consistency" icon="check-double">
    A unified visual language across all touchpoints ensures brand consistency and user familiarity.
  </Accordion>
  
  <Accordion title="Efficiency" icon="bolt">
    Pre-built components and patterns accelerate development by up to 60%, reducing time to market.
  </Accordion>
  
  <Accordion title="Accessibility" icon="universal-access">
    Every component meets WCAG 2.1 AA standards, ensuring inclusive experiences for all users.
  </Accordion>
  
  <Accordion title="Scalability" icon="chart-line">
    A flexible system that grows with your product needs, supporting themes and customization.
  </Accordion>
</AccordionGroup>

## Quick Start

Get up and running with TARMAC in minutes:

<Steps>
  <Step title="Install the package">
    ```bash
    npm install @tarmac/design-system
    ```
  </Step>
  
  <Step title="Import styles">
    ```tsx
    import '@tarmac/design-system/styles.css';
    ```
  </Step>
  
  <Step title="Use components">
    ```tsx
    import { Button } from '@tarmac/design-system';
    
    function App() {
      return <Button variant="primary">Get Started</Button>;
    }
    ```
  </Step>
</Steps>

## Who Uses TARMAC

TARMAC is designed for:

- **Designers** - Create consistent experiences with ready-to-use Figma components
- **Developers** - Build faster with pre-built, tested components
- **Product Teams** - Maintain brand consistency across all products
- **Stakeholders** - Ensure quality and accessibility standards
