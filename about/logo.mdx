# TARMAC Logo

The TARMAC logo is a key element of our brand identity. Use it consistently to maintain brand recognition.

## Logo Variations

### Primary Logo

<img src="/assets/brand/tarmac-logo-light.svg" alt="TARMAC primary logo" style={{ maxWidth: '300px', margin: '24px 0' }} />

The primary logo should be used in most contexts.

### Dark Version

<img src="/assets/brand/tarmac-logo-dark.svg" alt="TARMAC dark logo" style={{ maxWidth: '300px', margin: '24px 0', background: '#000', padding: '20px' }} />

Use the dark version on light backgrounds.

## Usage Guidelines

### Clear Space
Maintain clear space around the logo equal to the height of the "T" in TARMAC.

### Minimum Size
- **Digital**: 32px height minimum
- **Print**: 0.5 inch height minimum

### Color Variations

<Tabs>
  <Tab title="On Light Backgrounds">
    Use the dark version on light backgrounds for maximum contrast and readability.
  </Tab>
  
  <Tab title="On Dark Backgrounds">
    Use the light version on dark backgrounds to ensure visibility.
  </Tab>
  
  <Tab title="Monochrome">
    When color isn't available, use the black or white monochrome version.
  </Tab>
</Tabs>

## Don'ts

<Warning>
  Do not:
  - Rotate or distort the logo
  - Change the logo colors
  - Add effects like shadows or gradients
  - Place the logo on busy backgrounds
  - Recreate or modify the logo
  - Use old versions of the logo
</Warning>
