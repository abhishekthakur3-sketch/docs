## Overview

Logos are treated as system components, not static assets. Each product within the portfolio can have its own logo, but all logos follow a shared structure, sizing logic, and usage rules to ensure consistency, recognizability, and scalability across platforms.

## Logo Types

### Logo (Wordmark)

The full textual representation of the product or brand. Used when space allows and clear identification is required — ideal for headers, dashboards, and brand-forward surfaces.

### Logomark

A compact symbolic version derived from the wordmark. Used in constrained spaces such as navigation bars, app launchers, favicons, or dense UI layouts.

### Pictorial

An iconic visual mark representing the brand. Used for app icons, social media avatars, and contexts where the wordmark is too wide.

## Logo Sizes

| Size | Height | Usage |
|------|--------|-------|
| XXSmall | 14px | Micro contexts, dense data UI |
| XSmall | 16px | Favicons, compact nav, footer |
| Small | 24px | Navigation bars, sidebar headers |
| Medium | 28px | Standard headers, toolbars |
| Large | 34px | Dashboard headers, landing pages |
| XLarge | 48px | Hero sections, splash screens, marketing |

---

## Color Variants

| Variant | Background | Usage |
|---------|------------|-------|
| Black | Light backgrounds | Default for light mode UI |
| White | Dark backgrounds | Default for dark mode UI, overlays |

## Usage Guidelines

<CardGroup cols={2}>
  <Card title="✓ Do" icon="check" color="#16A34A">
    - Use the correct logo type for the context (wordmark for headers, logomark for nav)
    - Maintain clear space around the logo equal to the height of the mark
    - Use Black variant on light backgrounds, White on dark
    - Scale logos only to the defined size tokens
  </Card>
  <Card title="✗ Don't" icon="xmark" color="#DC2626">
    - Don't rotate, distort, or stretch the logo
    - Don't change logo colors beyond Black/White variants
    - Don't add effects like shadows, gradients, or outlines
    - Don't place logos on busy or low-contrast backgrounds
    - Don't recreate or modify the logo
    - Don't use outdated versions of the logo
  </Card>
</CardGroup>

---

## Clear Space and Minimum Size

<Warning>
  Maintain clear space around the logo equal to the height of the logomark. Never crowd the logo with other elements.
</Warning>

| Context | Minimum Height |
|---------|---------------|
| Digital (web/app) | 14px (XXSmall) |
| Print | 0.5 inch |
| Favicon | 16px × 16px |
