## Type Styles

### Display

High-impact, hero-level communication. Used for landing pages, feature announcements, and marketing surfaces.

| Weights | Bold (700), Semi Bold (600), Medium (500), Regular (400) |

### Body

The default reading styles for content, forms, and descriptive UI. B1 is the standard body size; B2 is used for dense layouts, tables, and secondary content.

<Tabs>
  <Tab title="Specs">

    |-------|------|-------------|----------------|-------------------|---------|
    | B1 | 16px | 24px | +0.3 (bold/semibold), +0.2 (medium/regular/light) | 0 | Default reading size — content, forms, descriptive UI |
    | B2 | 14px | 20px | +0.3 | 0 | Dense layouts, tables, labels, secondary content |

    Weight variants: **Bold** (700), **Semi Bold** (600), **Medium** (500, default), **Regular** (400), **Light** (300).

### Caption

Small text for helper hints, timestamps, metadata, and constrained UI. Use sparingly — captions should support, not carry, the content.


<Tabs>
  <Tab title="Specs">

    |-------|------|-------------|----------------|-------------------|---------|
    | C1 | 12px | 16px | +0.4 | 0 | Helper text, hints, timestamps, secondary metadata |
    | C2 | 10px | 16px | +0.4 | 0 | Rare use — constrained UI, legal text, technical notations |

    Weight variants: **Bold** (700), **Semi Bold** (600), **Medium** (500), **Regular** (400), **Light** (300).

## Line Height

Line height controls vertical rhythm and readability. TARMAC uses tighter line heights for large text and looser values for body content.

| Text Role | Font Size | Line Height | Ratio |
|-----------|-----------|-------------|-------|
| Caption (C1–C2) | 10–12px | 16px | 1.2–1.4× |
| Body (B1–B2) | 14–16px | 20–24px | 1.3–1.5× |
| Heading (H1–H5) | 20–40px | 26–52px | 1.2–1.4× |
| Display | 48px | 58px | 1.0–1.2× |

<Info>
  **Accessibility rules:**
  - Body text line height must not go below **1.45×** to maintain readability for users with cognitive disabilities.
  - Display text line height must not exceed **1.2×** to keep hero text visually cohesive.
</Info>

## Letter Spacing

Letter spacing (tracking) is adjusted per text role. Larger text uses negative tracking for visual tightness; smaller text uses positive tracking for legibility.

| Text Role | Font Size | Letter Spacing | Direction |
|-----------|-----------|----------------|-----------|
| Caption (C1–C2) | 10–12px | +0.4 to +0.6em | Looser — aids legibility at small sizes |
| Body (B1–B2) | 14–16px | +0.2 to +0.5em | Slightly open — comfortable reading |
| Heading (H1–H5) | 20–40px | -0.1 to -0.5em | Tighter — creates visual density |
| Display | 48px | -0.5 to -1em | Tightest — high-impact, compact feel |

<Warning>
  **Accessibility rules:**
  - Never use negative letter spacing on body or caption text — it reduces readability.
  - Ensure all text remains readable at **200% browser zoom**.
</Warning>
