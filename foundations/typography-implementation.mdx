## Accessibility

<AccordionGroup>
  <Accordion title="Contrast Requirements" icon="eye">
    **WCAG 2.1 Level AA**
    - Normal text (&lt; 18px): **4.5:1** contrast ratio minimum
    - Large text (≥ 18px or ≥ 14px bold): **3:1** contrast ratio minimum

    All typography color tokens in TARMAC are pre-validated against these thresholds for both light and dark modes. Disabled states are intentionally below contrast thresholds to signal non-interactivity.
  </Accordion>
  <Accordion title="Minimum Font Sizes" icon="text-height">
    - **Body text**: 16px (B1) minimum for primary content
    - **Secondary text**: 14px (B2) minimum for labels and supporting content
    - **Captions**: 12px (C1) minimum for metadata and helper text
    - **Constrained UI**: 10px (C2) only when absolutely necessary

    Always use `rem` units so text scales with user browser preferences.
  </Accordion>
  <Accordion title="Line Height and Spacing" icon="arrows-up-down">
    - Body text: line height ≥ **1.45×** (WCAG 1.4.12)
    - Paragraph spacing: at least **1.5×** the font size between paragraphs
    - Letter spacing: never negative for body or caption text

    These rules ensure readability for users with dyslexia and low vision.
  </Accordion>
  <Accordion title="Semantic HTML" icon="code">
    Always use proper heading elements (`h1`–`h5`) rather than styled `div` or `span` elements. Screen readers rely on heading structure for navigation.

    ```html
    <!-- Correct -->
    <h1>Page Title</h1>
    <h2>Section Heading</h2>
    <p>Body paragraph with <strong>emphasis</strong>.</p>

    <!-- Incorrect -->
    <div class="h1">Page Title</div>
    <span style="font-size: 34px">Section Heading</span>
    ```
  </Accordion>
  <Accordion title="Zoom and Reflow" icon="magnifying-glass-plus">
    - All text must remain readable at **200% browser zoom**
    - Text must reflow within the viewport — no horizontal scrolling
    - Avoid fixed-width containers that clip text at larger zoom levels
    - Use relative units (`rem`, `em`) instead of `px` for font sizes
  </Accordion>
</AccordionGroup>

## Code Examples

### CSS Custom Properties

```css
:root {
  /* Font Family */
  --font-family-primary: "Noto Sans", -apple-system, BlinkMacSystemFont,
                          "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif;

  /* Font Sizes */
  --text-caption-2: 0.625rem;  /* 10px */
  --text-caption-1: 0.75rem;   /* 12px */
  --text-body-2: 0.875rem;     /* 14px */
  --text-body-1: 1rem;         /* 16px */
  --text-h5: 1.25rem;          /* 20px */
  --text-h4: 1.5rem;           /* 24px */
  --text-h3: 1.75rem;          /* 28px */
  --text-h2: 2.125rem;         /* 34px */
  --text-h1: 2.5rem;           /* 40px */
  --text-display: 3rem;        /* 48px */

  /* Font Weights */
  --font-weight-light: 300;
  --font-weight-regular: 400;
  --font-weight-medium: 500;
  --font-weight-semibold: 600;
  --font-weight-bold: 700;

  /* Line Heights */
  --leading-caption: 16px;
  --leading-body-2: 20px;
  --leading-body-1: 24px;
  --leading-h5: 26px;
  --leading-h4: 32px;
  --leading-h3: 36px;
  --leading-h2: 44px;
  --leading-h1: 52px;
  --leading-display: 58px;

  /* Letter Spacing */
  --tracking-display: -0.05em;
  --tracking-h1: -0.5px;
  --tracking-h2: -0.25px;
  --tracking-h3: -0.15px;
  --tracking-h4: 0;
  --tracking-h5: 0;
  --tracking-body-bold: 0.3px;
  --tracking-body: 0.2px;
  --tracking-caption: 0.4px;
}
```

### TypeScript Tokens

```typescript
export const typography = {
  fontFamily: {
    primary: '"Noto Sans", -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif',
  },
  fontSize: {
    'caption-2': '0.625rem',  // 10px
    'caption-1': '0.75rem',   // 12px
    'body-2': '0.875rem',     // 14px
    'body-1': '1rem',         // 16px
    h5: '1.25rem',            // 20px
    h4: '1.5rem',             // 24px
    h3: '1.75rem',            // 28px
    h2: '2.125rem',           // 34px
    h1: '2.5rem',             // 40px
    display: '3rem',          // 48px
  },
  fontWeight: {
    light: 300,
    regular: 400,
    medium: 500,
    semibold: 600,
    bold: 700,
  },
  lineHeight: {
    caption: '16px',
    'body-2': '20px',
    'body-1': '24px',
    h5: '26px',
    h4: '32px',
    h3: '36px',
    h2: '44px',
    h1: '52px',
    display: '58px',
  },
  letterSpacing: {
    display: '-0.05em',
    h1: '-0.5px',
    h2: '-0.25px',
    h3: '-0.15px',
    h4: '0',
    h5: '0',
    bodyBold: '0.3px',
    body: '0.2px',
    caption: '0.4px',
  },
} as const;

// Typography color tokens
export const typographyColors = {
  display: {
    light: { primary: '#000000', secondary: '#1A1A1A', disabled: '#808080' },
    dark:  { primary: '#FFFFFF', secondary: '#F2F2F2', disabled: '#666666' },
  },
  heading: {
    light: { primary: '#0D0D0D', secondary: '#1A1A1A', disabled: '#949494' },
    dark:  { primary: '#F2F2F2', secondary: '#D9D9D9', disabled: '#666666' },
  },
  body: {
    light: { primary: '#1A1A1A', secondary: '#262626', disabled: '#B3B3B3' },
    dark:  { primary: '#E6E6E6', secondary: '#CCCCCC', disabled: '#666666' },
  },
  caption: {
    light: { primary: '#404040', secondary: '#333333', disabled: '#BDBCBC' },
    dark:  { primary: '#BFBFBF', secondary: '#BDBCBC', disabled: '#666666' },
  },
} as const;
```

<Info>Always test typography with real content across different devices and languages to ensure optimal readability. Noto Sans supports variable font features — use them for fine-grained weight control when available.</Info>
