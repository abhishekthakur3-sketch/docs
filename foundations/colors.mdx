# Colors

## Overview

The TARMAC Design System uses a carefully crafted color palette designed for clarity, accessibility, and visual harmony. Our color system provides semantic meaning, ensures sufficient contrast ratios, and supports both light and dark themes.

## Usage Guidelines

### Do's and Don'ts

<CardGroup cols={2}>
  <Card title="✓ Do" icon="check" color="#10b981">
    **Use semantic colors**
    - Green for success states
    - Red for errors and destructive actions
    - Yellow for warnings
    - Blue for informational content
    
    **Ensure sufficient contrast**
    - Text: ≥ 4.5:1 ratio
    - Large text: ≥ 3:1 ratio
    - UI components: ≥ 3:1 ratio
    
    **Use color tokens**
    - Reference design tokens
    - Maintain consistency
    - Enable theme switching
  </Card>
  
  <Card title="✗ Don't" icon="xmark" color="#ef4444">
    **Don't use color alone**
    - Always pair with text or icons
    - Don't rely on color for meaning
    
    **Don't use arbitrary colors**
    - Avoid hardcoded hex values
    - Don't create custom colors
    - Stick to the palette
    
    **Don't ignore contrast**
    - Test with accessibility tools
    - Verify WCAG compliance
    - Consider color blindness
  </Card>
</CardGroup>

## Accessibility Considerations

<AccordionGroup>
  <Accordion title="✓ WCAG Contrast Requirements" icon="eye">
    **Level AA (Minimum)**
    - Normal text (< 18px): 4.5:1 contrast ratio
    - Large text (≥ 18px or ≥ 14px bold): 3:1 contrast ratio
    - UI components and graphics: 3:1 contrast ratio
    
    **Level AAA (Enhanced)**
    - Normal text: 7:1 contrast ratio
    - Large text: 4.5:1 contrast ratio
    
    All TARMAC colors are designed to meet AA standards at minimum.
  </Accordion>
  
  <Accordion title="✓ Color Blindness" icon="eye-low-vision">
    **Considerations**
    - Don't rely on color alone to convey information
    - Use patterns, icons, or text labels alongside color
    - Test designs with color blindness simulators
    - Ensure sufficient contrast between adjacent colors
    
    **Common Types**
    - Deuteranopia (red-green, most common)
    - Protanopia (red-green)
    - Tritanopia (blue-yellow, rare)
  </Accordion>
  
  <Accordion title="✓ Testing Tools" icon="flask">
    - WebAIM Contrast Checker
    - Chrome DevTools Accessibility Panel
    - Stark (Figma/Sketch plugin)
    - Color Oracle (color blindness simulator)
    - axe DevTools
  </Accordion>
</AccordionGroup>

<Warning>Never use color as the only means of conveying information. Always pair color with text, icons, or patterns.</Warning>

## Dark Theme Support

<Info>TARMAC Design System supports dark theme by inverting the neutral scale and adjusting semantic colors.</Info>

```css
[data-theme="dark"] {
  /* Invert neutral scale */
  --color-neutral-50: #171717;
  --color-neutral-100: #262626;
  --color-neutral-200: #404040;
  --color-neutral-300: #525252;
  --color-neutral-400: #737373;
  --color-neutral-500: #a3a3a3;
  --color-neutral-600: #d4d4d4;
  --color-neutral-700: #e5e5e5;
  --color-neutral-800: #f5f5f5;
  --color-neutral-900: #fafafa;
  
  /* Adjust semantic colors for dark backgrounds */
  --color-success-600: #22c55e;
  --color-warning-600: #f59e0b;
  --color-error-600: #ef4444;
  --color-info-600: #3b82f6;
}
```
