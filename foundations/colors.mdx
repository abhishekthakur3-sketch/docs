---
title: Colors
description: Color system and palette for the TARMAC Design System
icon: "palette"
---

Color in the Tarmac Design System is built around a small set of core colors, extended into semantic tokens for surfaces, text, icons, and borders. The system separates raw values from semantic intent across three layers: Core → Semantic → Usage.

<Warning>Core colors are palette-only. No component or screen should reference them directly. All UI usage must go through semantic tokens.</Warning>
