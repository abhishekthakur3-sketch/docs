## Overview

All TARMAC components are fully keyboard accessible, following established WAI-ARIA interaction patterns. Users should be able to navigate, interact with, and operate every component without a mouse.

---

## Standard Keys

| Key | Action |
|-----|--------|
| Tab | Move focus to next interactive element |
| Shift + Tab | Move focus to previous interactive element |
| Enter | Activate buttons, links, submit forms |
| Space | Activate buttons, toggle checkboxes, select options |
| Escape | Close modals, dismiss overlays, cancel actions |
| Arrow Keys | Navigate within composite widgets (menus, tabs, radio groups) |
| Home / End | Jump to first/last item in a list or menu |

<Tabs>

## Focus Indicators

All interactive elements display a visible focus ring when navigated via keyboard:

| Property | Value |
|----------|-------|
| Style | 2px solid outline |
| Color | rgba(35, 150, 251, 0.4) — info blue |
| Offset | 2px from element edge |
| Shape | Follows element border-radius |

<Warning>
  Never remove focus indicators with `outline: none`. If you need to customize them, ensure the replacement meets 3:1 contrast ratio against adjacent colors.
</Warning>

---

## Focus Order

Focus order must follow the visual reading order of the page:

1. Skip link (hidden until focused)
2. Header / top navigation
3. Main content area (top to bottom, left to right)
4. Sidebar (if present)
5. Footer

<Tip>
  Use the Tab key to navigate through your page and verify the order feels logical. If focus jumps unexpectedly, check your DOM order and avoid positive `tabindex` values.
</Tip>

## Testing Checklist

| Check | Pass Criteria |
|-------|---------------|
| All interactive elements reachable | Tab reaches every button, link, input, and control |
| Focus order is logical | Matches visual layout, no unexpected jumps |
| Focus indicators visible | Clear ring on every focused element |
| No keyboard traps | User can always Tab away (except intentional traps in modals) |
| Escape closes overlays | Modals, dropdowns, and popovers close on Escape |
| Enter/Space activate controls | Buttons and toggles respond correctly |
| Arrow keys work in widgets | Tabs, menus, radio groups, sliders navigate with arrows |
| Skip link works | "Skip to main content" link appears on focus and works |
